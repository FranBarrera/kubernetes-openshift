Métodos de instalación

  • Oc cluster up
  • Minishift

oc cluster up

  • Instalar Docker
  • Configurar insecure registry y reiniciar docker 
    • /etc/sysconfig/docker 
     
    INSECURE_REGISTRY='--insecure-registry 172.30.0.0/16'
  • Descargar oc

    Github.com/openshift/origin/releases/latest

  • Ejecutar y eliminar
    •  # oc cluster up / # oc cluster down

Minishift

Herramienta creada como Minikube

  • Instalar Virtualbox, KVM, Hyperv...
  • Descargar Minishift

    • Github.com/jimmidyson/minishift/releases/latest 

     # minishift start / # minishift stop
